Zoektekst: Hoofdletterongevoelig?

Geef hierboven een tekst in en druk op de Highlight knop. Doet ie het of doet ie het niet? De spanning is te snijden! Als alles goed gaat zal de zoektekst gehighlight worden in deze instructies. Als je programma niet werkt, probeer dan eens te debuggen. Zet een breakpoint in je setup functie. Wordt die eigenlijk wel opgeroepen? Zoneen, is er iets mis met je window load event listener. Zoja, zet dan een breakpoint in je event listener function voor het click event van de button en klik op de button. Wordt je event listener opgeroepen? Zoneen, is er iets mis met het registreren van de listener. Ga stap voor stap door de code en kijk naar de waarden van de variabelen die je gebruikt, zijn ze zoals je ze verwacht had? Als je event listener wel wordt uitgevoerd is er wellicht iets mis met de code die voor de highlight moet zorgen. Oh wat spannend!
